寫出TABLE裡面放FROM表單語法
在 HTML 中，通常會將

放入 之中，用來整齊地排列標籤（Label）與輸入框（Input）。


VS Code 快速輸入技巧 (Emmet)
如果你想用 VS Code 的快捷鍵一次產生這個結構，可以在 HTML 檔輸入以下縮寫後按 Tab：
form>table>tr*2>td*2
重點說明：
包在最外面：這樣點擊按鈕時，表格內所有的輸入資料才會被一起送出。
label 與 id 對應：這對無障礙網頁與使用者體驗很有幫助（點擊文字也能選中輸入框）。
colspan="2"：這用來合併儲存格，讓按鈕可以置中橫跨兩欄。
